Catinder App

Description

  • Catinder Web App

  • https://catinder-app.web.app/

  • 03 Juni, 2021

A Web App that helps cat lovers find a partner for their pet cats to mate. This Catinder App has features such as a matching up feature with cat owners in the application, a geolocation API to find the whereabouts of users within a certain distance range, real-time chat for customers, user contact lists, profiles. This Catinder App application was developed by 4 people, including 2 Back-end developers, and 2 Front-end developers. I am a Front-end developer responsible for delivering all APIs or integrations from the Back-end (SERVER) to the Front-end which I will later process into data and display it, not only that I am also responsible for managing all layouts in the application.
