Sebuah aplikasi web app untuk mengelola data - data Pelamar, lowongan pekerjaan dan Proses Lowongan Pekerjaan di aplikasi tersebut. HRIS Career Web ini adalah aplikasi instansi lembaga pemerintahan yang dibawah naungan Badan Pendapatan Daerah (BAPENDA). Aplikasi HRIS Career Web ini di development oleh 3 orang, diantaranya 1 orang Back-end developer, 1 orang Fullstack Developer, dan 1 orang Front-end developer. Saya berposisi sebagai Front-end developer yang bertanggung jawab untuk mendeliver semua api atau integrasi dari Back-end (SERVER) ke Front-end yang nanti nya saya olah menjadi sebuah data dan menampilkannya, tidak hanya itu saya juga bertanggung jawab untuk mengelola layout pada HRIS Admin Web ini.
