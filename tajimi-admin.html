A Web App Admin Dashboard application that manages all food and beverage products entered into the application and will later be delivered to the Tajimingo Shop application to be displayed. Tajimi Admin Web App has features such as product management, order management, customer management, incoming payments, store management and payments entered into the application and there are features displayed with chart graphs. The Tajimi Admin Web App application was developed by 4 people, including 2 Back-end developers, and 2 Front-end developers. I am a Front-end developer who is responsible for delivering all APIs or integrations from the Back-end (SERVER) to the Front-end which I will later process into data and display it, not only that I am also responsible for managing all layouts in the application.
