An admin Dashboard web app application for managing employee data, human resources (HR), work jobs and financial management. This HRIS Admin Web is an application for government agencies under the auspices of the Regional Revenue Agency (BAPENDA). This HRIS Admin Web application was developed by 3 people, including 1 Back-end developer, 1 Fullstack Developer, and 1 Front-end developer. I am a Front-end developer responsible for delivering all APIs or integrations from the Back-end (SERVER) to the Front-end which I will later process into data and display it, not only that I am also responsible for managing the layout on this HRIS Admin Web.
